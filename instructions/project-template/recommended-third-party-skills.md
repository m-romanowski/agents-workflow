# Recommended Third-Party Skills Template

## Purpose

- template for an informational repository-local index of optional third-party skills
- keeps public recommendations separate from installed personal skill state

## Template Rules

- keep this file informational only
- do not imply that listed skills are required, installed, or loaded by default
- include only recommendations that are broadly useful to users of the workflow
- keep installation details high-level unless the repository explicitly documents a supported environment
- route to this file from `instructions/project/index.md` only when the operator asks about optional or recommended skills

## Suggested Entry Fields

- name
- purpose
- source URL
- supported agents or environments
- install model
- activation examples
- caveats
- recommendation level

## Example Notes

- recommendations should remain optional
- listed skills should not override repository workflow or safety rules
- personal installed-skill state belongs outside the repository unless the operator explicitly chooses otherwise
